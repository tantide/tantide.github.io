Hugo + Stack 搭建博客

2025-10-20
阅读时长: 2 分钟

使用Hugo和Stack主题搭建本地博客并通过GitHub Actions自动部署到GitHub Pages

在开始之前，必要的准备

  1. 创建GitHub账号，用于博客展示和托管站点文件。
  2. 安装git，用于推送站点文件到GitHub。
  3. 基础的Linux使用知识。

参考链接

https://brume.top/p/build-your-personal-blog-with-hugo
https://letere-gzj.github.io/hugo-stack/p/hugo/custom-blog
https://github.com/CaiJimmy/hugo-theme-stack-starter

环境说明

Ubuntu 24.04.3 LTS
git version 2.43.0
hugo v0.151.1-1cdd17882c28a9f23278d38ef03b403954cfb1e5+extended

本地建站

在命令行执行如下命令，注意替换 站点名称
hugo new site 站点名称

hugo默认不提供主题，所以需要自行导入主题，此处使用Jimmy Cai创作的Stack主题。

# 添加Stack主题并使用其提供的默认配置和示例
# 注意替换 站点名称 GitHub用户名 局域网IP
cd 站点名称
git init
git submodule add https://github.com/CaiJimmy/hugo-theme-stack/ themes/hugo-theme-stack
cp -r themes/hugo-theme-stack/exampleSite/content/* content/
cp themes/hugo-theme-stack/exampleSite/hugo.yaml ./
rm -f hugo.toml
rm -rf content/post/rich-content
sed -i 's|example.com|GitHub用户名.github.io|' hugo.yaml
hugo server -D --baseURL=http://局域网IP:1313 --bind=局域网IP

好了，现在访问 http://局域网IP:1313 查看效果。

GitHub自动部署

  • 在GitHub上创建名称为 站点名称 的仓库，用于存放站点源文件
  • 创建一个名称为 GitHub用户名.github.io 的public仓库，用于存放静态页面
  • 前往Setttings -> Developer Settings -> Personal access tokens，创建一个token(classic)
  • 将token保存到仓库的Settings -> Secrets and variables -> Actions
  • 在站点目录创建一个.github/workflows/deploy.yaml文件

# 注意替换 Token变量名 GitHub用户名

name: Deploy to Github Pages

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
        - name: Checkout
          uses: actions/checkout@v4
          with:
              submodules: recursive
              fetch-depth: 0

        - name: Setup Hugo
          uses: peaceiris/actions-hugo@v3
          with:
              hugo-version: "latest"
              extended: true

        - name: Build
          env:
            HUGO_ENV: production
          run: hugo --minify --gc

        - name: Deploy 🚀
          uses: peaceiris/actions-gh-pages@v4
          with:
              PERSONAL_TOKEN: ${{ secrets.Token变量名 }}
              EXTERNAL_REPOSITORY: GitHub用户名/GitHub用户名.github.io
              PUBLISH_BRANCH: main
              PUBLISH_DIR: ./public
              commit_message: auto deploy
